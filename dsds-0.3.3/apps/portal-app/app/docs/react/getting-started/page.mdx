# 시작하기

DSDS Portal에 오신 것을 환영합니다!

이 문서는 DSDS (Samsung DS Design System) React 컴포넌트 라이브러리 사용법을 안내합니다.

## 주요 기능

- **React 컴포넌트**: [Radix UI](https://www.radix-ui.com/) 기반의 고품질 컴포넌트
- **디자인 시스템**: 일관된 삼성 DS 디자인 언어
- **TypeScript 지원**: 완전한 타입 안전성
- **접근성**: WCAG 가이드라인 준수
