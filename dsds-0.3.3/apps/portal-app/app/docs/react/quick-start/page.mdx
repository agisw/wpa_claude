# 빠른 시작

DSDS 컴포넌트를 사용해서 첫 번째 애플리케이션을 만들어보세요.

## 기본 버튼 사용하기

```tsx
import { Button } from "@dsds/react-radix-ui"

function MyComponent() {
  return (
    <Button variant="primary" size="medium">
      클릭하세요
    </Button>
  )
}
```

## 레이아웃 구성하기

```tsx
import { Header, Lnb } from "@dsds/react-radix-ui"

function App() {
  return (
    <div className="flex h-screen flex-col">
      <Header logo="My App" theme="dark" />
      <div className="flex flex-1">
        <Lnb data={{ title: "메뉴", items: [] }} />
        <main className="flex-1 p-4">
          {/* 메인 콘텐츠 */}
        </main>
      </div>
    </div>
  )
}
```
