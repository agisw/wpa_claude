import { Button, SpinnerIcon } from "@dsds/react-radix-ui"

# 버튼 컴포넌트

DSDS Button 컴포넌트는 다양한 스타일과 크기 옵션을 제공합니다.

## 기본 사용법

<Button>기본 버튼</Button>

```tsx
import { Button } from "@dsds/react-radix-ui"

<Button>기본 버튼</Button>
```

## Variant 옵션

<Button variant="primary">Primary 버튼</Button>

### Primary
```tsx
<Button variant="primary">Primary</Button>
```

<Button variant="secondary">Secondary</Button>

### Secondary
```tsx
<Button variant="secondary">Secondary</Button>
```

### Warning / Danger

<Button variant="warning">Warning</Button>
<Button variant="danger">Danger</Button>

```tsx
<Button variant="warning">Warning</Button>
<Button variant="danger">Danger</Button>
```


### Ghost

<Button variant="ghost">Ghost</Button>

```tsx
<Button variant="ghost">Ghost</Button>
```

### GhostLink

<Button variant="ghostLink">GhostLink</Button>

```tsx
<Button variant="ghostLink">GhostLink</Button>
```

## 크기 옵션

<Button size="small">작은 버튼</Button>
<Button size="medium">중간 버튼</Button>
<Button size="large">큰 버튼</Button>

```tsx
<Button size="small">작은 버튼</Button>
<Button size="medium">중간 버튼</Button>
<Button size="large">큰 버튼</Button>
```

## 상태

### Disabled

<Button disabled>비활성 버튼</Button>

```tsx
<Button disabled>비활성 버튼</Button>
```

## 아이콘 사용

<div className="flex items-center gap-2">
  <Button icon={<SpinnerIcon />} iconOption="before">아이콘 앞</Button>
  <Button icon={<SpinnerIcon />} iconOption="after">아이콘 뒤</Button>
</div>

```tsx
import { Button, SpinnerIcon } from "@dsds/react-radix-ui"

<div className="flex items-center gap-2">
  <Button icon={<SpinnerIcon />} iconOption="before">아이콘 앞</Button>
  <Button icon={<SpinnerIcon />} iconOption="after">아이콘 뒤</Button>
</div>

```

## 이벤트 처리

```tsx
<Button onClick={() => console.log('클릭됨')}>
  클릭 이벤트
</Button>
```
