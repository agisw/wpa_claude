# 컴포넌트

DSDS React 컴포넌트 라이브러리에서 제공하는 모든 컴포넌트를 확인하세요.

## 기본 컴포넌트

### 버튼 (Button)
- 다양한 variant: primary, secondary, outline, ghost
- 크기 옵션: small, medium, large
- 상태: default, hover, active, disabled

### 입력 (Input)
- 텍스트 입력
- 패스워드 입력
- 검색 입력
- 숫자 입력

### 폼 (Form)
- 폼 컨테이너
- 폼 필드
- 유효성 검사
- 에러 메시지

## 레이아웃 컴포넌트

### 헤더 (Header)
- 로고 영역
- GNB 메뉴
- 유틸리티 버튼

### LNB (Left Navigation Bar)
- 계층형 메뉴 구조
- 선택 상태 관리
- 접기/펼치기 기능

### 그리드 시스템
- 반응형 그리드
- 플렉스박스 기반
- 다양한 레이아웃 옵션

## 사용 예시

각 컴포넌트의 자세한 사용법은 하위 메뉴에서 확인하실 수 있습니다.
