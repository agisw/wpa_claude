# 폼 컴포넌트

DSDS에서 제공하는 다양한 폼 관련 컴포넌트들을 알아보세요.

## Input 컴포넌트

### 기본 텍스트 입력
```tsx
import { Input } from "@dsds/react-radix-ui"

<Input placeholder="텍스트를 입력하세요" />
```

### 패스워드 입력
```tsx
<Input type="password" placeholder="패스워드" />
```

### 검색 입력
```tsx
<Input type="search" placeholder="검색어를 입력하세요" />
```

## Label 컴포넌트

```tsx
import { Label } from "@dsds/react-radix-ui"

<Label htmlFor="email">이메일</Label>
<Input id="email" type="email" />
```

## Form 컴포넌트

```tsx
import { Form, FormField, FormItem, FormLabel, FormControl, FormMessage } from "@dsds/react-radix-ui"

<Form>
  <FormField name="username">
    <FormItem>
      <FormLabel>사용자명</FormLabel>
      <FormControl>
        <Input placeholder="사용자명을 입력하세요" />
      </FormControl>
      <FormMessage />
    </FormItem>
  </FormField>
</Form>
```

## Select 컴포넌트

```tsx
import { Select, SelectContent, SelectItem, SelectTrigger, SelectValue } from "@dsds/react-radix-ui"

<Select>
  <SelectTrigger>
    <SelectValue placeholder="옵션을 선택하세요" />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="option1">옵션 1</SelectItem>
    <SelectItem value="option2">옵션 2</SelectItem>
    <SelectItem value="option3">옵션 3</SelectItem>
  </SelectContent>
</Select>
```

## Checkbox & Radio

### Checkbox
```tsx
import { Checkbox } from "@dsds/react-radix-ui"

<Checkbox id="terms" />
<Label htmlFor="terms">약관에 동의합니다</Label>
```

### Radio Group
```tsx
import { RadioGroup, RadioGroupItem } from "@dsds/react-radix-ui"

<RadioGroup>
  <div className="flex items-center space-x-2">
    <RadioGroupItem value="option1" id="r1" />
    <Label htmlFor="r1">옵션 1</Label>
  </div>
  <div className="flex items-center space-x-2">
    <RadioGroupItem value="option2" id="r2" />
    <Label htmlFor="r2">옵션 2</Label>
  </div>
</RadioGroup>
```
