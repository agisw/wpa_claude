# 변경 이력

이 프로젝트의 모든 주요 변경 사항이 이 파일에 기록됩니다.

형식은 [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)를 기반으로 하며,
이 프로젝트는 [Semantic Versioning](https://semver.org/spec/v2.0.0.html)을 준수합니다.

## [0.1.2] - 2025-09-25

### 신규 기능
- **Tabs 컴포넌트**: 상세한 문서화와 개선된 스타일을 포함한 탭 컴포넌트 강화
- **탭 스타일 시스템**: 키보드 네비게이션 및 접근성 기능을 지원하는 새로운 탭 스타일링 시스템

### 기능 개선
- **Header 컴포넌트**: 레이아웃 및 반응형 개선을 위한 헤더 스타일 정제
- **아이콘 유틸리티**: 아이콘 유틸리티 함수 개선 (GearIcon 개선 포함)
- **텍스트영역 스타일**: 시각적 일관성 향상을 위한 boxes.css의 텍스트영역 스타일 업데이트
- **Breadcrumb 컴포넌트**: 상세한 props 정의 및 문서화 강화
- **컴포넌트 문서화**: 사용 예제 및 props 설명을 포함한 문서화 개선
- **Storybook Stories**: 명확성과 구성 개선을 위한 Header 스토리 향상

### 버그 수정
- **코드 정리**: index.ts에서 주석 처리된 vuetify 스타일 import 제거
- **Tab 컴포넌트**: 오타 수정 및 탭 버튼 높이 조정
- **Header 검색박스**: 다크 테마용 비활성화 배경 및 테두리 색상 추가
- **QA 문서**: Tabs QA 문서의 섹션 제목 업데이트 및 시각적 테스트에 추가 스토리 포함
- **파일 정리**: 사용하지 않는 파일 및 폐기된 샘플 파일 제거

### 기술적 개선
- **시각적 스냅샷**: 시각적 회귀 테스트 스냅샷 업데이트
- **접근성**: 탭 컴포넌트의 접근성 기능 및 문서화 개선

## [0.1.1] - 2025-09-17

### 신규 기능
- **Radio 컴포넌트**: 새로운 라디오 버튼 컴포넌트 구현 및 스타일링
- **Toggle 컴포넌트**: 토글 스위치 컴포넌트 및 관련 스타일 구현
- **Tooltip 컴포넌트**: 툴팁 컴포넌트 강화 및 새로운 props 추가
- **Boxes 컴포넌트**: Textbox, Searchbox, Calbox 컴포넌트 기능 향상
  - CalendarIcon 및 MagnifyIcon 컴포넌트에 onClick prop 추가
  - TextboxWrapper 컴포넌트 스타일링 개선
- **HamburgerIcon 컴포넌트**: 새로운 햄버거 메뉴 아이콘 추가
- **Toast 스타일**: 커스텀 토스트 알림을 위한 스타일 시스템 추가

### 기능 개선
- **Header 컴포넌트**:
  - 네비게이션 로직 개선 및 메뉴 스타일 업데이트
  - 비활성화 상태 스타일 및 disabled prop 지원 추가
  - 호버 스타일 개선 (비활성화 상태에서만 적용)
- **Footer 컴포넌트**:
  - 새로운 스타일 및 아이콘 통합
  - 텍스트 및 버튼 스타일 패딩 조정
  - 구성 요소 정렬 옵션 개선
- **Button 컴포넌트**:
  - 패딩 및 스타일 개선
  - 새로운 alert 색상 변형 추가
  - 스피너 아이콘 조건부 표시 기능
- **Icon 컴포넌트들**:
  - GearIcon을 named function으로 변경 및 SVG 크기 조정
  - 아이콘 호버 스타일 접근성 개선
  - DummyIcon 및 HamburgerIcon에 data-testid와 aria-label props 추가

### 버그 수정
- **달력 아이콘**: 아이콘 표시 문제 수정
- **저작권 아이콘**: 선 굵기(stroke width) 수정
- **검색박스 데모**: 데모 기능 수정
- **텍스트박스**: 작은 크기 텍스트박스의 패딩 및 높이 조정으로 일관된 레이아웃 보장
- **툴팁**: TooltipContent 오프셋 조정으로 모든 방향에서 일관된 위치 설정
- **시각적 스냅샷**: QA 문서 및 시각적 회귀 테스트용 스냅샷 업데이트

### 문서화 개선
- **QA 문서**: Radio, Toggle, Tooltip, Boxes, Footer 컴포넌트용 QA 문서 추가
- **Storybook Stories**: 모든 새 컴포넌트에 대한 상세한 사용 예제 및 스토리 추가
- **컴포넌트 문서**: Calbox, Searchbox, Textbox 사용 모드 및 예제 명확화
- **한국어 지원**: 'Doing' 및 'Done' 상태에 대한 한국어 번역 추가

### 기술적 개선
- **시각적 회귀 테스트**: 새로운 컴포넌트들에 대한 포괄적인 시각적 테스트 추가
- **스타일 시스템**: 일관된 디자인 토큰 및 스타일 적용
- **접근성**: 아이콘 및 interactive 요소들의 접근성 속성 강화

## [0.1.0] - 2025-09-03

### 프로젝트 초기화
- **프로젝트 구조**: DSDS React Radix UI 컴포넌트 라이브러리 초기 설정
- **빌드 시스템**: Vite, TypeScript, Rollup 기반 빌드 환경 구축
- **Storybook**: 컴포넌트 문서화 및 개발 환경 설정
- **테스트 환경**: Playwright를 활용한 시각적 회귀 테스트 구축

### 핵심 컴포넌트 구현
- **Header 컴포넌트**:
  - 기본 헤더 레이아웃 및 네비게이션 기능
  - 탭 지원 기능 (withTabs 변형)
  - 검색박스 통합 및 스타일링
  - 로고 클릭 핸들링 (homeUrl, onLogoClick props)
- **Footer 컴포넌트**:
  - 커스터마이즈 가능한 푸터 아이템 및 테마 지원
  - 저작권 정보 표시 (CopyrightIcon 포함)
  - 기밀 정보 표시 (ConfidentialIcon 포함)
- **LNB (Left Navigation Bar) 컴포넌트**:
  - 트리 구조 네비게이션 지원
  - 검색 기능 및 경로 관리
  - 리사이징 기능 및 접근성 향상
  - 아코디언 방식 확장/축소 기능
- **Page 컴포넌트**:
  - 페이지 레이아웃 구조 및 헤더 통합
  - 동적 높이 계산 (useContainerHeight 훅)
  - 페이지 콘텐츠 영역 스타일링

### UI 기본 요소
- **Button 컴포넌트**:
  - 다양한 변형 및 상태 지원
  - 포커스 및 호버 스타일
  - 배지 알림 기능 (BadgeNotificationCount)
- **Input Box 컴포넌트들**:
  - Textbox: 기본 텍스트 입력 컴포넌트
  - Searchbox: 검색 기능이 포함된 입력 컴포넌트
  - Calbox: 달력 선택 기능이 포함된 입력 컴포넌트
  - TextboxWrapper: 공통 입력 요소 래퍼
- **Badge 컴포넌트**: 상태 및 알림 표시용 배지 시스템
- **Modal & Dialog**: 모달 및 대화상자 기본 구현
- **ScrollArea**: 커스텀 스크롤바를 포함한 스크롤 영역
- **Tabs**: 탭 네비게이션 기본 구현
- **Toast**: 알림 메시지 시스템 (Sonner 통합)

### 아이콘 시스템
- **기본 아이콘들**:
  - GearIcon, ExternalIcon, ConfidentialIcon, CopyrightIcon
  - LnbIcon, CalendarIcon, MagnifyIcon
  - HamburgerIcon (나중에 추가)
- **아이콘 프롭스**: 색상 커스터마이징 및 접근성 속성 지원

### 레이아웃 및 예제
- **Layout Examples**:
  - 다양한 페이지 레이아웃 템플릿
  - SDP SRS 상세 및 관리 레이아웃 샘플
  - 헤더와 탭이 있는 레이아웃 예제
- **Table 컴포넌트**: 데이터 테이블 기본 구현 및 페이지네이션
- **Navigation 컴포넌트**: 네비게이션 구조 및 컴포넌트들

### 스타일 시스템
- **DSDS 디자인 토큰**: 일관된 디자인 시스템 토큰 적용
- **CSS 모듈 시스템**: 컴포넌트별 스타일 모듈화

### 개발 도구 및 문서화
