# DSDS React Radix UI v0.1.2

## 새로운 기능

### Tabs 컴포넌트 강화
- **향상된 Tabs 컴포넌트**: 상세한 문서화와 개선된 스타일을 포함한 탭 컴포넌트
- **탭 스타일 시스템**: 키보드 네비게이션 및 접근성 기능을 지원하는 새로운 탭 스타일링 시스템

## 주요 개선사항

### Header 컴포넌트
- 레이아웃 및 반응형 개선을 위한 헤더 스타일 정제
- 다크 테마용 검색박스 비활성화 배경 및 테두리 색상 추가

### 아이콘 및 유틸리티
- **아이콘 유틸리티**: GearIcon 등 아이콘 유틸리티 함수 개선
- **텍스트영역 스타일**: 시각적 일관성 향상을 위한 boxes.css 업데이트

### 문서화 개선
- **Breadcrumb 컴포넌트**: 상세한 props 정의 및 문서화 강화
- **컴포넌트 문서화**: 사용 예제 및 props 설명을 포함한 문서화 개선
- **Storybook Stories**: 명확성과 구성 개선을 위한 Header 스토리 향상

## 버그 수정

- **코드 정리**: index.ts에서 주석 처리된 vuetify 스타일 import 제거
- **Tab 컴포넌트**: 오타 수정 및 탭 버튼 높이 조정
- **QA 문서**: Tabs QA 문서의 섹션 제목 업데이트 및 시각적 테스트에 추가 스토리 포함
- **파일 정리**: 사용하지 않는 파일 및 폐기된 샘플 파일 제거

## 기술적 개선

- **시각적 스냅샷**: 시각적 회귀 테스트 스냅샷 업데이트
- **접근성**: 탭 컴포넌트의 접근성 기능 및 문서화 개선
