# DSDS Vue Vuetify v0.2.1-beta (2025-10-17)

## 신규 기능

### 페이지 탭 시스템 확장
- **PageTabs 고급 컨트롤**: 드롭다운 탭 선택, 옵션 팝오버, 탭 닫기 확인 대화상자, Bulk Close 상태 등 데스크톱 워크플로에 맞춘 기능 추가

### 개발자 문서 툴링
- **CodeBlock & Markdown 컴포넌트**: Shiki 기반 하이라이트 및 스토리 문서 개선을 위한 Markdown/코드 블록 컴포넌트 추가

## 기능 개선
- **PageTabs UI/UX 개선**: 레이아웃 정리, 버튼 전환 애니메이션 개선, 스토리 전반의 옵션/확인 다이얼로그 예제 갱신
- **Storybook 문서**: Markdown 스토리 렌더링 품질 개선 및 예제 구조 정비

## 버그 수정
- PageTabs 확인 다이얼로그와 문서 렌더링 과정에서 발견된 표현 문제를 수정해 스토리 일관성을 확보
