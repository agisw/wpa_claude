# DSDS Vue Vuetify v0.1.1-alpha (2025-09-17)

## 새로운 기능

### 새로운 컴포넌트
- **VRadioGroup & VRadio**: 라디오 버튼 시스템
- **Pagination 시스템**: PaginationRoot, PaginationContent, PaginationNumber, PaginationDot
- **Header 시스템**: 13개 하위 컴포넌트로 구성된 완전한 헤더 시스템
- **CollapseToggleButton**: 접기/펼치기 토글 버튼
- **Toast**: 토스트 알림 시스템 및 showToast 함수

### VTooltip 개선
- 기본 툴팁 기능 구현
- 위치 자동 조정
- 다양한 방향 지원

## 주요 개선사항

- **Storybook 9.1.5** 업그레이드 및 성능 향상
- **디자인 토큰** 새로운 색상 시스템 추가
- **문서화** 컴포넌트 QA 문서 추가

## 버그 수정

- **VSelect 아이콘** 드롭다운 아이콘 디자인 일관성 개선
- **Header** disabled 상태에서 호버 효과 제거
- **VTooltip** 위치 조정 개선

## 개발 환경 개선

- **빌드 시스템** Vite 설정 최적화
- **Visual 테스트** 스냅샷 업데이트 및 회귀 테스트 추가
